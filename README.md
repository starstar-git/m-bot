使用步骤:
1. 将怪物图片放入 monsters 目录（PNG格式）
   - 可创建子目录作为怪物组
   - 同一怪物的不同状态放在同一组

2. 配置按键和参数:
   - 攻击键: 普通攻击
   - 技能键: 技能攻击（受冷却时间控制）
   - 移动键: 支持方向键和小键盘

3. 玩家识别:
   - 添加玩家角色截图到 player 目录
   - 程序自动获取玩家位置
   - 导入图片时会自动生成左右方向图片

4. 开始打怪:
   - 点击"开始打怪"按钮
   - 确保游戏窗口在前台

注意事项:
- 快捷键F5可以快速开始，在开始前请使用 测试玩家 、 测试怪物 按钮，查看是否是别到目标
- 使用HP/MP时，请先框选血条（100%血量所有红色区域，MP同理），检测并非百分百进准，更推荐使用宠物加血
- 截图区域设置后，提供更精准的检测区域，未设置则检测整个屏幕
- 识别阈值越高越精确，但可能漏检
- 技能冷却时间默认为9999秒
- 导入图片时会自动生成翻转图片（用于检测左右方向）
- 检测评率请根据自己电脑配置设置，请勿太高导致崩溃
